Task : Real estate agency with Spring Boot

Create Spring Boot application with CommandLineRunner and ApplicationRunner. This application should implement next business logic:

· Keep list of real estate objects (CRUD operations)

Increment counter of views for each real estate object

· Attach agent to each object

· Track all sold properties for each agent

· Provide report of top-5 agents by sum of sold properties for each month in time – range defined by the user

· Emulate price changing for each real estate object

· After running of application a few real estate transactions and price changes should happened in real-time with pre-defined schedule and logged on the console or in file

The application should contain:

1. Logger

2. DataSource bean

3. Order between two runners
